GeoGebra done right.


## Features

1. Extra tool keybindings.
2. Extra geometry functions.
3. Removed unnecessary dependencies and code.
4. XDG base directory support.
5. `.gsq` text/xml format. Reduced size. Easily searchable.
6. Modular macros.
